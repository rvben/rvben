### CLI Tools

All Rust, all installable via `cargo install` or `pip install`. Built for both humans and AI agents — every tool has `--json` output and a `schema` command for agent introspection.

* 🖥️ **[proxctl](https://github.com/rvben/proxctl)** — Manage Proxmox VE from your terminal. VMs, containers, nodes, storage, and declarative infrastructure with `apply`.
* 📡 **[unifi-cli](https://github.com/rvben/unifi-cli)** — Query UniFi Network controllers. Clients, devices, networks, events, and an interactive TUI dashboard.
* ⚡ **[n8nc](https://github.com/rvben/n8nc)** — n8n workflow automation. Sync workflows with Git, author locally, manage remote instances.
* 📒 **[yuki-cli](https://github.com/rvben/yuki-cli)** — Client for the Yuki bookkeeping SOAP API. Query administrations, find missing invoices, upload documents.
* 📦 **[upd](https://github.com/rvben/upd)** — Fast dependency updater for Python and Node.js projects.
